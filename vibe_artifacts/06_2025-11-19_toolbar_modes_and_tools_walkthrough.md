# Toolbar and Settings Update Walkthrough

## Changes

### Toolbar
- **Modes**: Simplified to "View" and "Edit".
    - **View**: Hides all creation tools. Allows dragging and panning.
    - **Edit**: Shows all creation tools (Box, Ball, Wall, Eraser, etc.).
- **Wall Tool**: Restored the "Draw Wall" tool to the toolbar in Edit mode.
- **Eraser**: "Eraser" is now available as a tool button in Edit mode.

### Settings Pane
- **Pause Button**: Moved the Pause/Play button to the top of the Settings Pane (side panel).

## Verification
### Manual Verification Steps
1.  **View Mode**:
    - Select "View" from the dropdown.
    - Verify that the tool buttons (Box, Ball, etc.) disappear.
    - Verify you can drag the view and objects (if applicable).
2.  **Edit Mode**:
    - Select "Edit" from the dropdown.
    - Verify that tool buttons appear.
    - Verify "Draw Wall" is present.
    - Select "Draw Wall" and verify you can draw walls.
    - Select "Eraser" and verify you can erase objects.
3.  **Pause Button**:
    - Open the Settings Pane (click the arrow if collapsed).
    - Click "Pause" and verify the simulation stops.
    - Click "Play" and verify the simulation resumes.
