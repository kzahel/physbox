# Project Overview: Physbox

Physbox is a 2D physics sandbox game built with TypeScript and Matter.js. It allows users to place various objects (boxes, balls, walls) and interactive elements (cars, monsters, robots) into a simulated world.

## Key Features
- **Physics Simulation**: Powered by Matter.js.
- **Interactive Tools**: Draw walls, erase objects.
- **Templates**: Pre-defined objects like Cars, Monsters, and a Player Robot.
- **Player Control**: A controllable robot character with movement and jetpack abilities.
- **Camera Control**: Pan and zoom capabilities.
